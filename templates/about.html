📋 What is FakeGuard?

FakeGuard is an intelligent web application that uses machine learning to automatically detect fake news. By analyzing text patterns, vocabulary, and linguistic features, the system can classify news articles as real or fake with high accuracy, helping users make informed decisions about the content they consume.
